表单布局

Bootstrap表单布局：

  • 垂直表单（默认）
  • 内联表单
  • 水平表单

垂直或基本表单

基本的表单结构是Bootstrap自带的，个别的表单控件自动接收全局样式。步骤：

  • 向父<form>元素添加role="form"
  • 把标签和控件繁杂一个带有class.form-group的<div>中。这是获取最佳间距所必需的。
  • 向所有的文本元素<input>、<textarea>和<select>添加class="form-control"

表单标签的使用

  • 首先为div添加class(.formgroup"堆叠表单"垂直方向)或.forminline"内联表单"水平方向
  • 随后创建label标签并赋值属性
  • 添加input标签，赋值class"form-control",赋值id(id为label属性)

表单控件

Bottstrap支持以下表单控件：

  • input
  • textarea
  • checkbox（复选框）
  • radio
  • select

一、Bootstrap Input

Bootstrap支持所有HTML5输入类型：text,password,datetime,datetime-local,date,month,time,week,number,email,url,search,tel,color

注意：如果input的type属性未经正确生命，输入框的样式将不会显示。
